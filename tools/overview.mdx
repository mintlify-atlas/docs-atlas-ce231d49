Kiosplay ships with three media conversion tools accessible directly from the admin panel at `/admin/tools`. You can convert video clips to animated GIFs, reduce video file sizes with H.264 compression, and extract individual PDF pages as PNG or JPEG images â€” all without leaving the interface. After conversion, each file can be downloaded or added straight to a kiosk playlist.

## How converted files are handled

After a conversion job finishes, the server returns a `jobId` and a download URL for each output file. You can either:

- **Download the file** directly from the tools panel.
- **Add the file to a kiosk playlist** by calling `POST /api/tools/add-to-kiosk` with the `jobId`, `filename`, and `kioskId`. The file is copied from the temporary directory into the kiosk's uploads folder and inserted into its playlist at the end of the sort order.

Temporary files are stored in `server/temp/<jobId>/`. To free up disk space, delete a job's files with:

```http
DELETE /api/tools/job/:jobId
```

You can also remove the directory manually. Temporary files are not cleaned up automatically on a schedule, so periodic maintenance is recommended on busy servers.
